### [3756\. 连接非零数字并乘以其数字和 II](https://leetcode.cn/problems/concatenate-non-zero-digits-and-multiply-by-sum-ii/)

难度：中等

给你一个长度为 `m` 的字符串 `s`，其中仅包含数字。另给你一个二维整数数组 `queries`，其中 <code>queries[i] = [l<sub>i</sub>, r<sub>i</sub>]</code>。

对于每个 `queries[i]`，提取 **子串** <code>s[l<sub>i</sub>..r<sub>i</sub>]</code>，然后执行以下操作：

- 将子串中所有 **非零数字** 按照原始顺序连接起来，形成一个新的整数 `x`。如果没有非零数字，则 `x = 0`。
- 令 `sum` 为 `x` 中所有数字的 **数字和**。答案为 <code>x &times; sum</code>。

返回一个整数数组 `answer`，其中 `answer[i]` 是第 `i` 个查询的答案。

由于答案可能非常大，请返回其对 <code>10<sup>9</sup> + 7</code> 取余数的结果。

**子串** 是字符串中的一个连续、**非空** 字符序列。

**示例 1：**

> **输入：** s = "10203004", queries = \[[0,7],[1,3],[4,6]]
> **输出：** [12340, 4, 9]
> **解释：**
>
> - `s[0..7] = "10203004"`
>   - `x = 1234`
>   - `sum = 1 + 2 + 3 + 4 = 10`
>   - 因此，答案是 <code>1234 &times; 10 = 12340</code>。
> - `s[1..3] = "020"`
>   - `x = 2`
>   - `sum = 2`
>   - 因此，答案是 <code>2 &times; 2 = 4</code>。
> - `s[4..6] = "300"`
>   - `x = 3`
>   - `sum = 3`
>   - 因此，答案是 <code>3 &times; 3 = 9</code>。

**示例 2：**

> **输入：** s = "1000", queries = \[[0,3],[1,1]]
> **输出：** [1, 0]
> **解释：**
>
> - `s[0..3] = "1000"`
>   - `x = 1`
>   - `sum = 1`
>   - 因此，答案是 <code>1 &times; 1 = 1</code>。
> - `s[1..1] = "0"`
>   - `x = 0`
>   - `sum = 0`
>   - 因此，答案是 <code>0 &times; 0 = 0</code>。

**示例 3：**

> **输入：** s = "9876543210", queries = \[[0,9]]
> **输出：** [444444137]
> **解释：**
>
> - `s[0..9] = "9876543210"`
>   - `x = 987654321`
>   - `sum = 9 + 8 + 7 + 6 + 5 + 4 + 3 + 2 + 1 = 45`
>   - 因此，答案是 <code>987654321 &times; 45 = 44444444445</code>。
>   - 返回结果为 <code>44444444445 \bmod  (10<sup>9</sup> + 7) = 444444137</code>。

**提示：**

- <code>1 <= m == s.length <= 10<sup>5</sup></code>
- `s` 仅由数字组成。
- <code>1 <= queries.length <= 10<sup>5</sup></code>
- <code>queries[i] = [l<sub>i</sub>, r<sub>i</sub>]</code>
- <code>0 <= l<sub>i</sub> <= r<sub>i</sub> < m</code>
