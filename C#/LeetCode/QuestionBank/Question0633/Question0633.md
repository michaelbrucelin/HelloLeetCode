### [633\. 平方数之和](https://leetcode.cn/problems/sum-of-square-numbers/)

难度：中等

给定一个非负整数 `c` ，你要判断是否存在两个整数 `a` 和 `b`，使得 <code>a<sup>2</sup> + b<sup>2</sup> = c</code>。

**示例 1：**

> **输入：** c = 5
> **输出：** true
> **解释：** 1 &times; 1 + 2 &times; 2 = 5

**示例 2：**

> **输入：** c = 3
> **输出：** false

**提示：**

- <code>0 <= c <= 2<sup>31</sup> - 1</code>
