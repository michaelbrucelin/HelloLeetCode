### [1895\. 最大的幻方](https://leetcode.cn/problems/largest-magic-square/)

难度：中等

一个 <code>k &times; k</code> 的 **幻方** 指的是一个 <code>k &times; k</code> 填满整数的方格阵，且每一行、每一列以及两条对角线的和 **全部** **相等**。幻方中的整数 **不需要互不相同**。显然，每个 <code>1 &times; 1</code> 的方格都是一个幻方。

给你一个 <code>m &times; n</code> 的整数矩阵 `grid`，请你返回矩阵中 **最大幻方** 的 **尺寸** （即边长 `k`）。

**示例 1：**

> ![](./assets/img/Question1895_01.jpg)
>
> **输入：** grid = \[[7,1,4,5,6],[2,5,1,6,4],[1,5,4,3,2],[1,2,7,3,4]]
> **输出：** 3
> **解释：** 最大幻方尺寸为 3。
> 每一行，每一列以及两条对角线的和都等于 12。
>
> - 每一行的和：5+1+6 = 5+4+3 = 2+7+3 = 12
> - 每一列的和：5+5+2 = 1+4+7 = 6+3+3 = 12
> - 对角线的和：5+4+3 = 6+4+2 = 12

**示例 2：**

> ![](./assets/img/Question1895_02.jpg)
>
> **输入：** grid = \[[5,1,3,1],[9,3,3,1],[1,3,3,8]]
> **输出：** 2

**提示：**

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 50`
- <code>1 <= grid[i][j] <= 10<sup>6</sup></code>
