二分法看成矩阵搜索就好理解多了，把数组展开成一个矩阵。例如对于[1,2,3,5]，展成的矩阵就是：

```c
1/1 2/1 3/1 5/1
1/2 2/2 3/2 5/2
1/3 2/3 3/3 5/3
1/5 2/5 3/5 5/5
```

再省略掉数值>=1的一侧：

```c
1/2
1/3 2/3
1/5 2/5 3/5
```

arr[j]为分母，arr[i]为分子。对于mid=0.5：

- j=1时搜索第一行，count+=0；
- j=2时搜索第二行，count+=1；
- j=3时搜索第三行，count+=2。

最后count == 3 == k，搜索过程中更新的最大结果2/5即为最终答案。
