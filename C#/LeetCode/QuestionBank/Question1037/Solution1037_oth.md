这个题就是已知三个顶点坐标来计算三角形面积是否为$0$： 面积公式$S=\dfrac{(x_1y_2-x_2y_1)+(x_2y_3-x_3y_2)+(x_3y_1-x_1y_3)}{2}$，去个常数项，提个公因子答案就来了。

```java
class Solution {
    public boolean isBoomerang(int[][] p) {
        
        return (p[0][0] * (p[1][1] - p[2][1]) + p[1][0] * (p[2][1] - p[0][1]) + p[2][0] * (p[0][1] - p[1][1])) != 0;
    }
}
```
