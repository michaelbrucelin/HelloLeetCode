### [3393\. 统计异或值为给定值的路径数目](https://leetcode.cn/problems/count-paths-with-the-given-xor-value/)

难度：中等

给你一个大小为 <code>m &times; n</code> 的二维整数数组 `grid` 和一个整数 `k`。

你的任务是统计满足以下 **条件** 且从左上格子 `(0, 0)` 出发到达右下格子 `(m - 1, n - 1)` 的路径数目：

- 每一步你可以向右或者向下走，也就是如果格子存在的话，可以从格子 `(i, j)` 走到格子 `(i, j + 1)` 或者格子 `(i + 1, j)`。
- 路径上经过的所有数字 `XOR` 异或值必须 **等于** `k`。

请你返回满足上述条件的路径总数。

由于答案可能很大，请你将答案对 <code>10<sup>9</sup> + 7</code> **取余** 后返回。

**示例 1：**

> **输入：** grid = \[[2, 1, 5], [7, 10, 0], [12, 6, 4]], k = 11
> **输出：** 3
> **解释：**
> 3 条路径分别为：
>
> - <code>(0, 0) &rarr; (1, 0) &rarr; (2, 0) &rarr; (2, 1) &rarr; (2, 2)</code>
> - <code>(0, 0) &rarr; (1, 0) &rarr; (1, 1) &rarr; (1, 2) &rarr; (2, 2)</code>
> - <code>(0, 0) &rarr; (0, 1) &rarr; (1, 1) &rarr; (2, 1) &rarr; (2, 2)</code>

**示例 2：**

> **输入：** grid = \[[1, 3, 3, 3], [0, 3, 3, 2], [3, 0, 1, 1]], k = 2
> **输出：** 5
> **解释：**
> 5 条路径分别为：
>
> - <code>(0, 0) &rarr; (1, 0) &rarr; (2, 0) &rarr; (2, 1) &rarr; (2, 2) &rarr; (2, 3)</code>
> - <code>(0, 0) &rarr; (1, 0) &rarr; (1, 1) &rarr; (2, 1) &rarr; (2, 2) &rarr; (2, 3)</code>
> - <code>(0, 0) &rarr; (1, 0) &rarr; (1, 1) &rarr; (1, 2) &rarr; (1, 3) &rarr; (2, 3)</code>
> - <code>(0, 0) &rarr; (0, 1) &rarr; (1, 1) &rarr; (1, 2) &rarr; (2, 2) &rarr; (2, 3)</code>
> - <code>(0, 0) &rarr; (0, 1) &rarr; (0, 2) &rarr; (1, 2) &rarr; (2, 2) &rarr; (2, 3)</code>

**示例 3：**

> **输入：** grid = \[[1, 1, 1, 2], [3, 0, 3, 2], [3, 0, 2, 2]], k = 10
> **输出：** 0

**提示：**

- `1 <= m == grid.length <= 300`
- `1 <= n == grid[r].length <= 300`
- `0 <= grid[r][c] < 16`
- `0 <= k < 16`
