#### 贡献值

找出每一对不同字符，然后看看这对不同的字符可以贡献多少个结果。

##### 示例
例如`s = "abbab", t = "bbbbb"`
```C
s[0] != t[0]，左侧可扩展0位，右侧可扩展2位，贡献了(0+1)*(2+1) = 3
s[0] != t[1]，左侧可扩展0位，右侧可扩展2位，贡献了(0+1)*(2+1) = 3
s[0] != t[2]，左侧可扩展0位，右侧可扩展2位，贡献了(0+1)*(2+1) = 3
s[0] != t[3]，左侧可扩展0位，右侧可扩展1位，贡献了(0+1)*(1+1) = 2
s[0] != t[4]，左侧可扩展0位，右侧可扩展0位，贡献了(0+1)*(0+1) = 1
s[1], s[2]在t中找不到不同的字符
s[3] != t[0]，左侧可扩展0位，右侧可扩展1位，贡献了(0+1)*(1+1) = 2
s[3] != t[1]，左侧可扩展1位，右侧可扩展1位，贡献了(1+1)*(1+1) = 4
s[3] != t[2]，左侧可扩展2位，右侧可扩展1位，贡献了(2+1)*(1+1) = 6
s[3] != t[3]，左侧可扩展2位，右侧可扩展1位，贡献了(2+1)*(1+1) = 6
s[3] != t[4]，左侧可扩展2位，右侧可扩展0位，贡献了(2+1)*(0+1) = 3
s[4]在t中找不到不同的字符

结果为：33
```
