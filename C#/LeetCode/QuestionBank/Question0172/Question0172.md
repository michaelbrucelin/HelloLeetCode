### [172\. 阶乘后的零](https://leetcode.cn/problems/factorial-trailing-zeroes/)

难度：中等

给定一个整数 `n`，返回 `n!` 结果中尾随零的数量。

提示 <code>n! = n &times; (n - 1) &times; (n - 2) &times; ... &times; 3 &times; 2 &times; 1</code>

**示例 1：**

> **输入：** n = 3
> **输出：** 0
> **解释：** 3! = 6，不含尾随 0

**示例 2：**

> **输入：** n = 5
> **输出：** 1
> **解释：** 5! = 120，有一个尾随 0

**示例 3：**

> **输入：** n = 0
> **输出：** 0

**提示：**

- <code>0 <= n <= 10<sup>4</sup></code>

**进阶：** 你可以设计并实现对数时间复杂度的算法来解决此问题吗？
